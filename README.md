
# Data Analytics & Business Intelligence - Task 1

## Project Overview

This project focuses on:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL Analysis
* Dashboard Creation using Power BI
* Business Insight Generation

## Tools Used

* Excel
* SQL (SQLite)
* Power BI

## Project Tasks

### 1. Data Cleaning

* Removed duplicates
* Handled missing values
* Standardized date formats
* Corrected text formatting

### 2. Exploratory Data Analysis

Created Pivot Tables for:

* Sales by Region
* Profit by Category
* Monthly Sales Trends
* Top Customers
* Top Products

### 3. SQL Analysis

Performed:

* GROUP BY analysis
* Sales aggregation
* Profit analysis
* Customer analysis
* Discount impact analysis

### 4. Dashboard Creation

Built an interactive Power BI dashboard with:

* KPI Cards
* Sales trends
* Region analysis
* Filters and slicers

## Key Insights

* Identified top-performing regions
* Found most profitable product categories
* Analyzed monthly sales trends
* Evaluated discount impact on profit
